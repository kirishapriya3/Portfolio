# 🌐 Kirisha Priya – Portfolio Website

A modern, responsive **full-stack developer portfolio** built using React and Tailwind CSS.
This portfolio showcases my projects, skills, and contact details with smooth animations and clean UI.

## 🛠️ Tech Stack

* ⚛️ React.js
* 🎨 Tailwind CSS
* 🧠 JavaScript (ES6+)
* 🌐 HTML5 & CSS3
* 🔗 Git & GitHub

## ✨ Features

* ✅ Fully responsive design (mobile + desktop)
* ✅ Smooth animations and transitions

1. Clone the repository

```
git clone https://github.com/your-username/your-repo-name.git
```

2. Navigate to project folder


3. Install dependencies

```

```

4. Run the project

```
npm run dev
```
